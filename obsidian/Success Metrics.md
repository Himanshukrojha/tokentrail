# Success Metrics

## V0 learning metrics (not vanity features)

| Metric | Target |
| ------ | ------ |
| Real prompts benchmarked | 50 in first month |
| Human agreement (golden set) | ≥ 70% |
| Time to recommend | < 2s |
| Caching correctly identified | ≥ 80% on repeated-prompt set |
| Runs logged | 100% |
| Rule explainability | Every output cites rule IDs |

## V1 metrics

| Metric | Target |
| ------ | ------ |
| Logged runs | 200+ |
| Override rate | < 25% (rules good enough) |
| Per-category accuracy | ≥ 75% |

## V2 metrics

| Metric | Target |
| ------ | ------ |
| ML rerank lift | +5% agreement vs V1 |
| Shadow-mode disagreement | tracked before rollout |

## Anti-metrics (ignore for now)

- GitHub stars
- Model count supported
- Image render aesthetics

Focus: **would a developer trust this recommendation for a real agent prompt?**
